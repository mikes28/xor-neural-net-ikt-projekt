# XOR neurális háló – IKT Projektmunka II.

Ez a projekt egy egyszerű neurális hálót valósít meg Pythonban, amely megtanulja az XOR logikai műveletet.  
Célja, hogy bemutassa, hogyan tanul egy modell: hogyan csökken a hiba, és hogyan változik a háló kimenete a tanulás során.

## Követelmények

- Python 3.x
- numpy
- matplotlib

- thee
